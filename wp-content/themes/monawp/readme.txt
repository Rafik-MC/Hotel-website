== Description ==

Mona is designed to have fast loading speed and low amount of HTTP requests. It has lots of layout, color, typography options and over 150 customizer presets. The theme works with major pagebuilders like Elementor, Spectra, Beaver Builder, Kadence blocks, SiteOrigin etc. Mona includes a horizontal menu in topbar, header and footer, sticky sidebar widget, sticky header, post and page meta options, breadcrumbs, related posts and SEO optimization. It also supports popular plugins like Contact Form 7, Infinite Scroll in Jetpack, Mailchimp for WordPress, Easy Table of Contents, WooCommerce & more.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= Does this theme support any plugins? =

MonaWP includes support for Infinite Scroll in Jetpack, Jetpack social share, Easy Table of Contents, Mailchimp for Wordpress, and others.
